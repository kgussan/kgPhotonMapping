GPUフォトンマッピングプログラム。


----------------------------------------------------------------
注意・特記事項
----------------------------------------------------------------
・ログは/log ディレクトリに出力される。

・GPUの種類によって動作が不安定になる。特にintel チップが不安定。
　原因としてはphoton samplingのシェーダ実行中に落ちる。
　この処理が長すぎるのかもしれない。
　場合によってはGPUの処理分割単位を変更すると動作するかもしれない。

・hlslはオフラインコンパイルしている。(windows sdkによるfxc.exe)
　実行時のコンパイルはなぜか失敗する。
　（これはdirectx11 SDK のコンパイラが指定されているのか？）

・実行時、GPUだと動作が不安定になることが確認されている。
　おそらくphotonSampling シェーダが遅すぎるためか。
　ドライバが落ちる。インテルのドライバはとても不安定。
　NVIDIAのものも落ちる場合がある。AMDでは動作を確認。
　この不安定さを緩和するため、photonSamplingを分割して
　実行できるようにしている。標準で４分割。

・他、お気づきの点あれば上記メールアドレスまでお気軽に連絡願います。
